# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.0.x | ✅ 支持 |

## 报告安全漏洞

如果你发现了安全漏洞，**请不要通过公开的 GitHub Issue 报告**。

请通过以下方式私下联系我们：

- GitHub 私信：发送消息给仓库维护者
- 邮件：联系仓库维护者

请在报告中包含：

- 漏洞类型（如 XSS、注入、认证绕过等）
- 影响范围和严重程度
- 复现步骤
- 可能的修复建议

## 响应时间承诺

| 阶段 | 时间 |
|------|------|
| 确认收到报告 | 48 小时内 |
| 初步评估和回复 | 7 天内 |
| 发布修复 | 视严重程度，尽快发布 |

感谢你帮助保护 ObClipper 的安全。
